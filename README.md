#  AERIA – Gemini-Powered Intelligent Assistant

## 🚀 Overview
AERIA is a modern AI chatbot powered by the Gemini API, designed to deliver intelligent, real-time responses with a premium user experience.

## ✨ Features
- ⚡ Real-time streaming responses
- 🎨 Glassmorphism UI + Dark Mode
- 🧠 Neural Backup System (Offline fallback)
- 📝 Custom Markdown rendering engine
- 💾 Session persistence using Local Storage

## 🛠️ Tech Stack
- HTML5, CSS
- JavaScript 
- Google Gemini API
